# 整体分层架构（Layered Architecture Overview）

> 本文档拆分自「团队 AI 协作平台分层架构设计 v2.0」§1
>
> 背景与设计目标请参阅 [00-背景与目标](00-background-and-goals.md)。

> **TL;DR**
> - 系统分为 Layer 0（外部系统）到 Layer 6（输出动作）共 7 层，职责清晰分离
> - 从 Meego Webhook / 飞书群消息摄入，经意图识别、记忆召回、上下文组装，到 Swarm 编排和 Sidecar 执行
> - 每层标注了对应的详细设计文档编号，便于按需深入

│   - 控制面(调度)与数据面(执行)通过 stdin/stdout (stream-json NDJSON) 通信                     │
